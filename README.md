# AI Land Price Predictor 🏠
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/juby1996/land-price-predictor/blob/main/land_price_predictor.ipynb)
## Project Overview
This project was developed during the **Upcode AI Bootcamp (April 2026)**. It features a Machine Learning model built with Python to predict land prices based on historical data.

## 🛠️ Tools & Technologies
* **Language:** Python 
* **Libraries:** Pandas (Data Manipulation), Scikit-Learn(Machine Learning Model)
* **Model:** Linear Regression

### 📊 Dataset
The model is trained on `land_prices.csv`, which contains historical pricing data including coordinates (latitude/longitude) and distances to key landmarks.
